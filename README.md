# ubuntu-bash

Update Maschine

```bash
sudo apt-get update -y && sudo apt-get upgrade -y
```

Installing Git

```bash 
sudo apt-get install git -y
```

Installing Zip

```bash
sudo apt-get install git-all -y
```

Installing zsh

```bash
sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"
```
